Requisitos:
1. Sistema Operativo: Windows
2. Java Runtime Environment instalado: 
	https://www.oracle.com/java/technologies/javase-jre8-downloads.html
3. VPN-FEUP ligado p/ DB.

Configuração do IP do Servidor OPC:
1. No ficheiro 'configOPC.txt' coloca-se o endereço IP do servidor OPC.
	Caso seja local, colocar: '127.0.0.1'.

Configuração da DB:
1. Remota: http://dbm.fe.up.pt/phppgadmin/
2. PostgreSQL;
3. Obrigatória a ligação VPN-FEUP para uso da BD.

Sequência de inicialização:
1. Abertura do SFS;
2. Arranque do programa Codesys;
3. Arranque do MES.
